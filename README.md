
## How to use this:
- install pow, node, npm
- `npm install`
- `echo 9999 > ~/.pow/node-sandbox`
- `nodemon app.coffee`
- open [http://node-sandbox.dev/](http://node-sandbox.dev/) in browser
- to run in production mode locally run `NODE_ENV=production PORT=9999 coffee app.coffee`


## Used HTML5 tech:
- Blob constructing
- FileReader API
- fileWriter API
- filesystem API
- file saver  API
- Drap/Drop
